# Лабораторная работа по теории информации

## Шифрование текста

GUI-приложение для шифрования и дешифрования текста двумя алгоритмами:
- **Шифр Плейфейра** — текст на английском языке
- **Алгоритм Виженера** (прогрессивный ключ) — текст на русском языке

## Запуск

```bash
python main.py
```

## Требования

- Python 3.8+
- Tkinter (обычно входит в поставку Python)

### Установка Tkinter (если отсутствует)

**macOS (Homebrew):**
```bash
brew install python-tk@3.14
```

**Ubuntu/Debian:**
```bash
sudo apt-get install python3-tk
```

**Windows:** Tkinter обычно уже установлен с Python.

## Использование

1. Выберите алгоритм (Плейфейр или Виженер)
2. Введите ключ
3. Введите текст вручную или загрузите из файла (кнопка «Загрузить из файла...»)
4. Выполните действие: **Шифровать** или **Дешифровать**
5. Результат отображается в поле «Результат»
6. При необходимости сохраните результат в файл

Программа игнорирует все символы, не являющиеся буквами выбранного алфавита.
